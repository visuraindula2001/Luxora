Luxora – Scalable Full-Stack E-Commerce System

🔗 Live Demo
https://luxora-tau-six.vercel.app  

📦 Repository
https://github.com/visuraindula2001/luxora  

Luxora is a full-stack e-commerce application built using the MERN stack.  
The project focuses on scalability, clean architecture, and modern backend engineering practices.

🛠️ Tech Stack

Frontend
- React.js

Backend
- Node.js
- Express.js

Database
- MongoDB

Tools & Technologies
- Docker
- GitHub Actions (CI/CD)
- Apache Kafka (event-driven architecture)
- Nginx (load balancing - simulated)
